# 🍌 Muz Animasyonu

10 saniyelik MP4 video animasyonu - dönen sarı muz ve metin.

## Kurulum

```bash
pip install moviepy pillow numpy
```

## Çalıştırma

```bash
python create_banana_animation.py
```

## Çıktı

- **banana_animation.mp4** - 10 saniyelik video
- Çözünürlük: 1280x720
- Metin: "Ben çok faydalı bir yiyeceğim"

## Özellikler

✅ Dönen muz animasyonu  
✅ Metinli alt yazı  
✅ MP4 formatı  
✅ 30 FPS hızında çalışma
